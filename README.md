05 os

05 os es un “sistema operativo” experimental hecho en turbowarp, diseñado para ejecutarse encima de windows (y otras plataformas) como una aplicación ligera, rápida y funcional para pcs modestas.

no intenta reemplazar a windows. intenta no estorbar.

qué es (y qué no)

no es un sistema operativo real

no es un entorno linux completo

no es una demo inútil para abrir 2 minutos

sí es:

un entorno tipo os hecho en bloques

una plataforma extensible con apps

usable en hardware viejo o limitado

pensada para arrancar rápido y hacer lo básico bien

características principales

arranque rápido (~3 segundos)

interfaz simple, sin ventanas reales

terminal funcional con comandos reales

sistema de apps instalables por link

chat global usando cloudlink

navegador integrado

reproductor de música

calculadora simple

gestor de fondos de pantalla

sistema de apagado y reinicio

copia de chat y logs al portapapeles

sistema de apps

las apps se distribuyen como .sprite3 o assets compatibles

se instalan desde un link directo

las apps se registran internamente por su url

actualmente las apps no permanecen al cerrar y abrir el projecto exportado
